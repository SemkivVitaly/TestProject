# Подключение Mission Planner через ESP32-S3-Box (MAVLink)

## Схема

```
Mission Planner (ПК)  ←→  Wi‑Fi (TCP или UDP)  ←→  ESP32-S3-Box  ←→  UART (пины 13/14)  ←→  Автопилот
```

Плата пересылает **сырые байты** (в т.ч. кадры MAVLink). Разбор MAVLink и отображение телеметрии делает Mission Planner.

## Шаг 1: Подключение автопилота к ESP32

- **TX автопилота** (телеметрия) → **GPIO 14** (RX ESP32)
- **RX автопилота** → **GPIO 13** (TX ESP32)
- **GND** → **GND**

В `config.h` задаётся скорость: `UART_BAUD` (часто **57600** или **115200** — должна совпадать с портом телеметрии на автопилоте).

## Шаг 2: Wi‑Fi

- Включите плату; в режиме **MODE_AP** она создаёт точку доступа.
- Подключите ПК к сети **ESP32S3_NEW** (пароль в `config.h`, по умолчанию `12345678`).
- В режиме AP IP платы: **192.168.2.1**.

## Шаг 3: Mission Planner

### Вариант A — TCP (рекомендуется)

1. В Mission Planner: **Connection Type** → **TCP**.
2. В поле подключения укажите: **192.168.2.1:8880** (или только IP, порт 8880).
3. Нажмите **Connect**.
4. После подключения в MP должны пойти телеметрия, HUD, параметры — данные с автопилота идут по UART на ESP32 и по Wi‑Fi в MP.

### Вариант B — UDP

1. **Connection Type** → **UDP**.
2. Порт: **14550**.
3. **Connect**. Mission Planner отправит пакеты на плату; плата запомнит адрес и будет слать ответы автопилота туда.

## Если в Mission Planner нет данных

- Проверьте проводку UART (TX→RX, RX→TX, GND).
- Убедитесь, что **UART_BAUD** в `config.h` совпадает со скоростью порта телеметрии на автопилоте (часто 57600 для телеметрии).
- В MP после Connect подождите 2–5 секунд — автопилот может начать слать поток после первого heartbeat от GCS.
- Откройте Serial Monitor платы (115200) — при подключении TCP должен появиться лог вида `TCP client 192.168.2.xxx`.

## Порты в config.h

| Параметр            | Значение | Назначение                          |
|---------------------|----------|-------------------------------------|
| SERIAL_TCP_PORT     | 8880     | Порт TCP для Mission Planner        |
| SERIAL_UDP_PORT     | 14550    | Порт UDP (стандарт MAVLink GCS)      |
| UART_BAUD           | 115200   | Скорость UART к автопилоту (или 57600) |
| SERIAL_RXPIN / TXPIN| 14, 13   | Пины UART на ESP32                   |
